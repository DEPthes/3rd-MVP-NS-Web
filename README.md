## 🐑 N력 키우기
> **N력(상상력)을 키울 수 있는 놀이터**
<br>

## 💻 주요 기능
- **나였다면**: 매일 새로운 주제를 제공하여 창의적인 아이디어를 표현할 수 있는 공간
- **N 보고서**: 다양한 주제에 대해 사용자들의 상상력을 분석하고, 상상력의 트렌드를 한눈에 파악할 수 있는 공간
- **N력 랭킹**: 나의 상상력이 어디에 위치하는지 확인하고, 다른 사용자들과의 랭킹을 비교할 수 있는 공간
<br>
